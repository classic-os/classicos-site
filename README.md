## What Is Classic OS?

**Classic OS is the economic operating system for Ethereum Classic.**

Four modules organized by economic intent:
- **Produce** - Create capital through mining (Mining OS)
- **Portfolio** - Observe capital and positions (DeFi dashboard)
- **Deploy** - Allocate capital into strategies (DeFi automation)
- **Markets** - Access liquidity and stablecoins (DEX + Brale)

**Unique value:** Transform global energy into yield-bearing assets on the only mature PoW chain with smart contracts.
